Peamised funktsioonid

Lucidchart on visuaalne koostööplatvorm, mis võimaldab kiiresti luua voogdiagramme, UML-skeeme, ERD-sid (andmebaasid) ja traatmuudeleid (wireframes).

Microsoft Project

History

Microsoft Project was created by Microsoft and first released in 1984, originally developed for DOS. It was one of the first project management software products for personal computers. The tool was created to help project managers plan, track and manage projects more efficiently. Development has continued ever since, and today it is available both as a desktop app and a cloud-based service (Microsoft Project Online).

Main Features

  • 📅 Gantt Charts — visualise project timelines and task dependencies
  • 👥 Resource Management — assign and track team members and their workload
  • 📊 Reporting — built-in reports for progress, costs and performance
  • 🔗 Microsoft 365 Integration — works with Teams, Excel and SharePoint
  • ☁️ Cloud Access — available online via Microsoft Project Online

Screenshots

Microsoft Project remains one of the most widely used project management tools in the world, especially in large enterprises. It supports both Waterfall and Agile methodologies, making it versatile for different types of projects.

ProjectLibre

Ajalugu

  • 👤 Lõid: Marc O'Brien, Laurent Chretienneau, Larry Kaye
  • 📅 Asutatud: 2012
  • 💡 Miks: Tasuta avatud lähtekoodiga alternatiiv Microsoft Projectile
  • ✅ Arendus jätkub: Jah — ProjectLibre Cloud on arendamisel

Peamised funktsioonid

  • 📅 Gantt-diagrammid — ülesannete, ajajoonte ja sõltuvuste visualiseerimine (FS, SS, FF, SF)
  • 👥 Ressursside haldus — meeskonnaliikmete määramine ja töökoormuse jälgimine
  • 📁 MS Projectiga ühilduv — avab ja salvestab .mpp faile
  • 📊 WBS ja RBS vaated — töö- ja ressursijaotuse struktuuridiagrammid
  • 💰 Eelarve jälgimine — kulude haldus baastaseme ja tegelike kulude võrdlusega
  • 🖥️ Platvormideülene — töötab Windowsis, macOS-is ja Linuxis (Java-põhine)

Ekraanipildid

ProjectLibre on üks enim allalaaditud avatud lähtekoodiga projektihalduse tööriistu maailmas. See toetab klassikalist veekose planeerimist ning on laialdaselt kasutusel hariduses, väikeettevõtetes ja mittetulundusühingutes tasuta alternatiivina Microsoft Projectile. Tulev ProjectLibre Cloud toob platvormile reaalajas koostöö võimaluse.
